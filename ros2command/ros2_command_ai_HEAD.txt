## motion
ROS2: topic_pub /motion_manager/task_cmd motion_manager_msgs/msg/TaskCmd
Example:
ros2 topic pub --once /motion_manager/task_cmd motion_manager_msgs/msg/TaskCmd '{task_type: 0, command_type: 0, motion_id: 1}'
Optional parameters:
- blocking_type (str)
Option conditions:
- [task_type] 0 (condition: MOTION_TASK) (parameters: motion_id: int, required=true)
- [task_type] 4 (condition: LEGACY_MOTION_TASK) (parameters: legacy_id: int, required=true)
- [task_type] 5 (condition: Pause preset path) (parameters: motion_id: int, required=true)
- [task_type] 6 (condition: Preset path) (parameters: preset_id: int, required=true)
- [command_type] 0 (condition: default start)
- [command_type] 2 (condition: only when task_type == 5) (parameters: task_type: int, required=true)
- [command_type] 4 (condition: stop all)
- [blocking_type] NONE (condition: resolve on completion)
- [blocking_type] SOFT (condition: resolve on completion)
- [blocking_type] HARD (condition: resolve on completion)
- [blocking_type] LOOP (condition: re-executes until release)
- [blocking_type] WAIT (condition: waits blocked-action release)

## sound_tts
ROS2: topic_pub /sounds_topic sounds_interfaces/msg/SoundsEvent
Example:
ros2 topic pub --once /sounds_topic sounds_interfaces/msg/SoundsEvent "{main_type: 1, sub_type: 6, text_value: 'hello', number_value: 0}"
Optional parameters:
- none
Option conditions:
- [command] tts (condition: sound list type == tts or not found)

## sound_music_url

Example:
ros2 topic pub --once /music_play_url std_msgs/msg/String "data: 'https://www.soundhelix.com/examples/mp3/SoundHelix-Song-1.mp3'"
Optional parameters:
- none
Option conditions:
- [command] music (condition: sound list type == music)
