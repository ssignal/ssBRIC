## micom_charger_control
ROS2: service_call /amr/micom_manager/charger_cmd amr_interfaces/srv/MicomRequest
Example:
ros2 service call /amr/micom_manager/charger_cmd amr_interfaces/srv/MicomRequest "{data: 'on'}"
Optional parameters:
- none
Option conditions:
- [command] true (condition: charger on) (parameters: req.data: str, required=true)
- [command] false (condition: charger off) (parameters: req.data: str, required=true)
