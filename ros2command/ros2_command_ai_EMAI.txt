# ROS2 Command Example List
# module: EMAI
# source: prompt_ros2

## vla_start
ROS2: topic_pub /emai/command std_msgs/msg/String
Example:
ros2 topic pub --once /emai/command std_msgs/msg/String "data: '{\"to\":\"inference\",\"data\":\"start\",\"model\":\"hamper\",\"msg\":\"folding_towel\"}'"
Optional parameters:
- model (str)
Option conditions:
- none

## emai_motion
ROS2: topic_pub /emai/command std_msgs/msg/String
Example:
ros2 topic pub --once /emai/command std_msgs/msg/String "data: '{\"to\":\"motion\",\"data\":\"start\",\"msg\":\"wave\"}'"
Optional parameters:
- blocking_type (str) - Resolve behavior
Option conditions:
- [blocking_type] NONE (condition: resolve on completion)
- [blocking_type] SOFT (condition: resolve on completion)
- [blocking_type] HARD (condition: resolve on completion)
- [blocking_type] LOOP (condition: re-executes motion until stop/release)
- [blocking_type] WAIT (condition: waits for blocked-action release after completion)

## teleoperation
ROS2: topic_pub /emai/command std_msgs/msg/String
Example:
ros2 topic pub --once /emai/command std_msgs/msg/String "data: '{\"to\":\"teleop\",\"data\":\"start\"}'"

- none
Option conditions:
- [data] start (condition: begin teleoperation)
- [data] stop (condition: end teleoperation)
