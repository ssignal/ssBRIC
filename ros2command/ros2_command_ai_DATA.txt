# ROS2 Command Example List
# module: DATA
# source: prompt_ros2

## robot_command
ROS2: topic_pub /rodam/robot/command std_msgs/msg/String
Example:
ros2 topic pub --once /rodam/robot/command std_msgs/msg/String "data: '{\"command\":\"boot\",\"data\":{},\"metadata\":{}}'"
Optional parameters:
- namespace (str) - default '/rodam'
- data (json) - optional command payload
- metadata (json) - optional metadata payload
Option conditions:
- [command] boot (condition: allowed when type == robotCommand)
- [command] shutdown (condition: allowed when type == robotCommand)
