# ROS2 Command Example List
# module: AMRConveyor
# source: prompt_ros2

## conveyor_reliable_action
ROS2: service_call /amr/charge/control or /amr/conveyor/control amr_interfaces/srv/MicomRequest
Example:
ros2 service call /amr/conveyor/control amr_interfaces/srv/MicomRequest "{data: 'FullTrayLoad'}"
Optional parameters:
- none
Option conditions:
- [command] charge (condition: service=/amr/charge/control) (parameters: req.data: str, required=true)
- [command] stop-charge (condition: service=/amr/charge/control) (parameters: req.data: str, required=true)
- [command] full-tray-load (condition: service=/amr/conveyor/control) (parameters: req.data: str, required=true)
- [command] full-tray-unload (condition: service=/amr/conveyor/control) (parameters: req.data: str, required=true)
- [command] empty-tray-discharge (condition: service=/amr/conveyor/control) (parameters: req.data: str, required=true)
